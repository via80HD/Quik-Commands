Brave repository error when trying to run ‘sudo apt update’ that says’ “…doesn’t support architecture ‘i386’ error” Gotta add arch=amd64 to repo list file. 1 . run: cd /etc/apt/sources.list.d/ 2. run: ls 3. Find your matching source list file & type the following command: sudo nano /etc/apt/sources.list.d/ 4. On the screen you are taken to, you will need to click the right arrow 5 times then type the following & dont forget the space at the end after 4 arch=amd64 5. Should look like the following at the front: deb* [arch=amd64* signed……. 6. Save it with ctrl+x, then type ‘y’ & click enter. 7. run sudo apt update & should be working now.
How to see if the PC is even seeing your USB/HDD (This is for when a usb isnt even showing when plugged in.) 1. Unplug the device you are testing for 2. Run the following command ls /dev/ | grep sd 3. Plug your device back in & re run the same command 4. If there is more lines after the 2nd time you ran it, that means the PC is seeing your USB. If the number of lines turns out to be the exact same as the first, that means the PC is not seeing your USB & will need to investigate further.
