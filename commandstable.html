Install YT-DLP (YouTube Down Loader) 1: sudo add-apt-repository ppa:tomtomtom/yt-dlp 2: sudo apt update 3: sudo apt install yt-dlp
Download Audio Only Using YT-DLP yt-dlp -f ‘ba’
